The Bird Hunter game includes smooth mouse-based shooting controls, multiple bird types with different speeds, real-time score updates, sound effects for shooting and hits, and a clean and responsive design. It is fully browser-based and easy to run on any modern web browser.

## Technologies Used
-HTML5
-CSS
-JavaScript

## How to Play
Open the game in your browser and click the Start Game button. Use your mouse to aim at the flying birds and click to shoot them. Each successful hit will increase your score. Try to shoot as many birds as possible before the timer runs out and achieve the highest score.
